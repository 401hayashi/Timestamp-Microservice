Microservicio de Marca de Tiempo

Convierte entre marcas de tiempo Unix y fechas UTC - freeCodeCamp

Descripción

Este microservicio convierte entre marcas de tiempo Unix y fechas en formato UTC. Proporciona una API que acepta una fecha como parámetro y devuelve un objeto JSON con ambas representaciones.

Endpoint: GET /api/:date?

El parámetro date puede ser una cadena de fecha en formato ISO-8601 (ej: 2016-11-20) o una marca de tiempo Unix (ej: 1479663089000).

Si el parámetro está vacío, se devolverá la fecha y hora actual.

Ejemplos de uso

GET /api/2015-12-25

{"unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT"}

GET /api/1451001600000

{"unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT"}

GET /api/invalid-date

{"error": "Invalid Date"}
